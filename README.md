# Luminis — Site Vitrine

Site de présentation Luminis, création de sites web pour artisans.

## Déploiement GitHub Pages

1. Upload les fichiers dans ce dépôt
2. Va dans **Settings → Pages**
3. Source : **Deploy from a branch** → branche `main` → dossier `/ (root)`
4. Clique **Save**
5. Ton site sera en ligne sur `https://TONPSEUDO.github.io/NOMDUREPOT`
